# Twiiter Blockchain Clone

A clone of Twitter social media platform using blockchain technologies such as web3 and smart contracts. 


## Tech Stack

- Next.js
- Tailwind CSS (for styling)
- Sanity CMS
- Solidity
- [Alchemy](https://www.alchemy.com/)
